1. Kinematics & Volumes (Eq 3.1 - 3.6)

The displacements of the piston ($x$) and displacer ($y$) drive the volume changes in the expansion ($V_e$) and compression ($V_c$) spaces.

$$x = \frac{x_0}{2}(1 + \cos(\theta - \varphi))$$ $$V_c = V_{d-c} + \frac{V_{S-P}}{2}(1 + \cos(\theta - \varphi)) + \frac{V_{S-D}}{2}(1 - \cos\theta)$$ $$\frac{dV_c}{dt} = -\frac{V_{S-P}}{2}\omega \sin(\theta - \varphi) + \frac{V_{S-D}}{2}\omega \sin\theta$$

Code Logic: These values dictate the geometry of the engine at every time step $t$.

2. Heat Transfer Coefficients (Eq 3.29 - 3.33)

Convective heat transfer is heavily dependent on piston velocity and instantaneous pressure.

$$h = 2.43 V_p^{1/3} (p_{inst} T_g)^{1/2}$$ $$dQ_H = h_H A_{h-H} (T_H - T_e)$$

Code Logic: Absolute values are enforced in JavaScript (`Math.abs`) to prevent `NaN` errors when calculating fractional powers of velocities.

3. Pressure Derivative (Eq 3.12)

The core of the 2nd-order model. The global pressure fluctuates based on the net heat transfer and volume changes.

$$\frac{dp}{dt} = \frac{1}{C_v V_T} \left( R(dQ_H + dQ_r + dQ_C) - C_p \left( p_c \frac{dV_c}{dt} + p_e \frac{dV_e}{dt} \right) \right)$$

Code Logic: Integrated using Euler's method: p = p + (dpdt * dt).

4. Mass Flows & State (Eq 3.10, 3.15, 3.18)

Mass is pushed between the chambers. The Ideal Gas Law establishes the new temperatures.

$$\dot{m}_{c-r} = -\frac{1}{R T_{c-r}} \left( -\frac{R}{C_p} dQ_C + p_c \frac{dV_c}{dt} + \frac{V_c}{\gamma} \frac{dp}{dt} \right)$$ $$\frac{dm_c}{dt} = -\dot{m}_{c-r} \quad \implies \quad m_c = m_c + \left(\frac{dm_c}{dt} \times dt\right)$$ $$T_c = \frac{p_c V_c}{R m_c}$$

Code Logic: Limits (`Math.max`) are applied to mass values in the JS to ensure mass never drops below zero during large iteration steps.

5. Work & Power Output (Eq 3.34 - 3.61)

Work is the integration of pressure over volume. Power is Work multiplied by Frequency.

$$W_i = \int_0^t \left( p_c \frac{dV_c}{dt} + p_e \frac{dV_e}{dt} \right) dt$$ $$P_i = W_i \times f$$
